# Проверка идеи

Прототип помогает увидеть слабые места раньше, чем проект станет дорогим. Мы собираем первый рабочий контур и оставляем место для быстрых итераций.

## Результат

- сценарии экранов;
- базовые компоненты;
- понятный текст интерфейса;
- список следующих улучшений.

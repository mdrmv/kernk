# Данные, которым можно доверять

Инженер оценивает схему подключения, состояние приборов, корректность показаний и возможность удалённого съёма данных. Это помогает увидеть пики потребления и спорные начисления.

## На выходе

- проверка узлов учёта;
- рекомендации по модернизации;
- подготовка данных для анализа;
- снижение риска ошибок в начислениях.

# Публикация без ручной рутины

После настройки достаточно отправить изменения в `main`, а GitHub Actions соберёт Jekyll-сайт и выложит свежую версию.

## Результат

- workflow для Pages;
- корректный `baseurl`;
- production-сборка;
- понятная точка входа для будущих правок.
